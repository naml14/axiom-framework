⚡ Axiom Demo Launcher

Dos flujos separados para evitar confusión: estático puro (abre directo) y SSR (requiere servidor).

🧱 Demo Estático Puro

No depende de rutas SSR. Puedes abrirlo como archivo local o servirlo con cualquier servidor estático.

Abrir static.html

🧪 Demo SSR

Renderizado del lado servidor usando el endpoint /ssr. Solo funciona cuando el demo server está corriendo.

Abrir /ssr

Tip: para SSR usa bun run demo y abre http://localhost:3000.
